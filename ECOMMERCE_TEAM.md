# 🛒 AI电商选品团队

## 团队配置（6人）

```
📊 选品分析师 (3人)
├── trend-spotter        趋势捕捉：各平台热搜/飙升词/蓝海品类
├── product-evaluator    四维评分：市场×竞争×利润×趋势
└── competitor-analyst   竞品拆解：差评挖掘/差异化机会/定价策略

🎙️ 内容创作师 (2人)
├── script-writer        带货话术生成：短视频脚本+直播话术
└── visual-planner       视觉策划：拍摄角度/场景搭建/封面设计

💰 运营策略师 (1人)
└── ops-manager          测款计划/投流策略/定价与利润模型
```

## 每日工作流程

```
🕘 08:00 选品早报
  ① 搜索各平台今日热搜品类
  ② 四维评分筛选 Top5 新品
  ③ 推送选品报告到手机

🕒 15:00 竞品快报
  ④ 监测对标达人新上商品
  ⑤ 追踪竞品价格/佣金变动
```

## 选品四维模型

选品评分 = 市场需求×30% + 竞争程度×25% + 利润空间×25% + 趋势热度×20%

| 维度 | 数据源 | 合格线 |
|------|--------|--------|
| 市场需求 | 搜索量/7天销量 | 搜索≥3000，销量≥5000 |
| 竞争程度 | 竞品数/头部集中度 | 竞品<500，CR3<40% |
| 利润空间 | 毛利率≥30%，佣金≥20% | 推广费≤15% |
| 趋势热度 | 搜索增长率/内容热度 | 增长率≥15% |

## 黄金品类（新手）

1. 小件生活百货（低退货、易拍摄）
2. 网红厨房收纳（视觉展示强）
3. 桌面/车载好物（客单价低决策快）
4. 宠物用品（高复购）
5. 节日应季品（粽子/月饼/暖宝宝）
